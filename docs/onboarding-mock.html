4 · Enforce
9:41●●● Wi‑Fi

One master switch

At the top of the main screen. Leave it on so rules apply while you use other apps.

Enforce rules On = apply per-app rules. Off = pause and restore system rotation.
Modes at a glance
  • Landscape / Portrait — lock a fixed orientation
  • Auto (sensor) — turn system auto-rotate on for that app
  • System default — no override
